# useTitle

设置文档的标题

### 代码演示

```javascript
import { useTitle } from '../use-title'

export default {
  setup() {
    const title = 'useTitle'
    useTitle(title)
  }
}
```

### 参数

| 参数  | 说明                            | 类型     |
| ----- | ------------------------------- | -------- |
| title | useTitle 要设置文档的标题字符串 | _string_ |
